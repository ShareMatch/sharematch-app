Strict Guidelines & Canonical Tokens

⚠️ This is the Source of Truth. Cursor must follow these rules strictly.

1. Required Rules

  1. Strict Token Usage: Always use brand.* tokens. Never use hex codes.
  2. Typography: Always use Inter. Playfair Display is FORBIDDEN.
  3. Component Primacy: Always use the defined patterns.
  4. Radius Rules: Button/Input = rounded-full, Card = rounded-xl.
  5. No Inline Styles: Use Tailwind classes only.
  6. No Arbitrary Tailwind: Use bg-gray-200 for inputs, not hex.
